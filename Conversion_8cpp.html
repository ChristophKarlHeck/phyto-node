Converts raw ADC measurements into analog voltage values.

Converts raw ADC byte inputs into analog voltage values.

Parameters
byte_inputsA vector of 3-byte arrays representing raw ADC measurements.
databitsThe number of bits used for ADC resolution (e.g., 8388608 for 24-bit ADC).
vrefThe reference voltage of the ADC in volts.
gainThe gain applied to the ADC measurements.
Returns
A vector of converted voltage values in millivolts.

This function processes raw ADC data represented as 3-byte arrays, calculates the corresponding analog voltage based on the ADC's resolution, reference voltage, and gain, and returns the values in millivolts.
