Converts raw ADC byte inputs into analog voltage values.

This function processes raw ADC data and converts it into a vector of corresponding analog voltage values based on the given ADC configuration.

Parameters
byte_inputsA vector of 3-byte arrays representing raw ADC data.
databitsThe number of data bits used by the ADC (e.g., 12, 16, 24).
vrefThe reference voltage used by the ADC in volts.
gainThe gain applied to the ADC readings.
Returns
A vector of floats representing the converted analog voltage values.
Note
This function assumes the input data is properly formatted and scaled according to the ADC's resolution and configuration.
