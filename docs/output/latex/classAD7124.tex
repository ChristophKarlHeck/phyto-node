\doxysubsection{Detailed Description}
Singleton class for interfacing with the \mbox{\hyperlink{classAD7124}{AD7124}} using SPI. 

The \mbox{\hyperlink{classAD7124}{AD7124}} class provides methods for initializing and interacting with the \mbox{\hyperlink{classAD7124}{AD7124}} Analog-\/to-\/\+Digital Converter (ADC) via SPI. It supports configuration of ADC channels, reading voltage data, and resetting or controlling the device.
