Converts raw ADC byte inputs into analog voltage values. 

This function processes raw ADC data and converts it into a vector of corresponding analog voltage values based on the given ADC configuration.

\begin{DoxyNote}{Note}
This function assumes the input data is properly formatted and scaled according to the ADC\textquotesingle{}s resolution and configuration.
