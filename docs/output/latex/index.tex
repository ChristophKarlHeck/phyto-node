 Phyto\+Node is a modular system for collecting, processing, and transmitting sensor data using an ADC connected to a microcontroller. Built on Mbed OS, it enables real-\/time operations and communicates seamlessly with a Raspberry Pi for advanced data processing.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md2}{}\doxysection{$<$strong$>$\+Project Overview$<$/strong$>$}\label{index_autotoc_md2}
The system comprises the following core components\+:
\begin{DoxyEnumerate}
\item {\bfseries{ADC Module}}\+: Interfaces with the \mbox{\hyperlink{classAD7124}{AD7124}} ADC for high-\/precision analog-\/to-\/digital conversion.
\item {\bfseries{Inter-\/\+Thread Communication}}\+: Manages data flow between threads using Mbed OS\textquotesingle{}s {\ttfamily Mail} objects.
\item {\bfseries{Serial Communication}}\+: Ensures efficient data transfer to a Raspberry Pi using Flat\+Buffers serialization.
\item {\bfseries{Utility Functions}}\+: Provides tools for data conversion and system performance monitoring.
\end{DoxyEnumerate}

\item {\bfseries{ADC Module}}\+:
\begin{DoxyItemize}
\item Manages data acquisition from the \mbox{\hyperlink{classAD7124}{AD7124}} ADC.
\item Configures channels and performs continuous readings.
\end{DoxyItemize}
\item {\bfseries{Interfaces}}\+:
\begin{DoxyItemize}
\item Implements a {\ttfamily \mbox{\hyperlink{classReadingQueue}{Reading\+Queue}}} for inter-\/thread communication using a singleton pattern.
\end{DoxyItemize}
\item {\bfseries{Serial Communication}}\+:
\begin{DoxyItemize}
\item Serializes ADC data into Flat\+Buffers format and transmits it over UART.
\end{DoxyItemize}
\item {\bfseries{Utilities}}\+:
\begin{DoxyItemize}
\item Converts raw ADC data to meaningful voltage values.
\item Monitors memory and CPU usage for performance optimization.
