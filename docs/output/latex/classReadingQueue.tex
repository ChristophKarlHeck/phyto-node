\doxysubsection{Detailed Description}
Singleton class for managing inter-\/thread communication using a mailbox. 

The {\ttfamily \mbox{\hyperlink{classReadingQueue}{Reading\+Queue}}} class provides a thread-\/safe mechanism for inter-\/thread communication using an Mbed OS Mailbox. It stores and manages messages containing ADC readings for multiple channels.
