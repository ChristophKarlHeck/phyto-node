\doxysubsection{Detailed Description}
Singleton class responsible for serializing and sending mail data over a serial port. 

The {\ttfamily \mbox{\hyperlink{classSerialMailSender}{Serial\+Mail\+Sender}}} class provides functionality to serialize ADC readings and transmit them over a serial connection. It ensures efficient and thread-\/safe communication using a singleton design pattern.
